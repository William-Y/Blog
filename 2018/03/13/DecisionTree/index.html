Decision Tree

It’s a study note during the learning process of Algorithm of Machine Learning

The model to be introduced is based on the genesis version of Decision Tree.–ID3 Decision Tree.

Here are some critical points of this process.

  • Information Entropy
  • Information Gain

We will introduce these concepts at the steps in need.

- Brief Flow Path

For a given data set, we assume that all the value of its features are discrete for simplyfying. Now, we build a root node that represent the whole data set. To enable us making better decision, we need to find a feature that split the data set into a number of pieces based on its values. More than that, the feature we’ve chosen should make those small datasets bias enough for us to make decision. For those small pieces which is still not bias enough, we need to repeat this preocess on them till they are bias enough or the features are used up.
